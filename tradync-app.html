Sincroniza MT4/MT5 con TradyncApp
Descarga el Expert Advisor para tu plataforma. Se instala en 2 minutos y tus operaciones se registran solas.
PASO 1 — COPIA EL ARCHIVO
Copia el archivo a MetaTrader
1. Abre MetaTrader 5
2. Menú Archivo → Abrir carpeta de datos
3. Entra en MQL5 → Experts
4. Copia TradyncSync_MT5.mq5 ahí
5. En MT5 → Navigator → clic derecho → Actualizar
6. Doble clic en el EA → MetaEditor → pulsa F7 para compilar
PASO 2 — ACTIVA WEBREQUESTS
Permite la conexión (solo una vez)
1. MT5: Herramientas → Opciones → Expert Advisors
2. Activa ✅ "Permitir WebRequest para las siguientes URL"
3. Pulsa + y añade:
https://www.tradyncapp.com
4. Pulsa Aceptar
PASO 3 — ARRASTRA AL GRÁFICO
Activa el EA
1. En Navigator → Expert Advisors → TradyncSync_MT5
2. Arrástralo a cualquier gráfico
3. Pestaña "Common" → activa ✅ "Allow WebRequests"
4. Pulsa OK — ¡listo!
🔄
Verificando...
Asegúrate de que el EA está activo y el botón "Algo Trading" está en verde en MT5.
